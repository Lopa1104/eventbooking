== Description ==
Use 
https://{site_url}/wp-json/my-api/v1/events/show

https://{site_url}/wp-json/my-api/v1/events/show/{2023-08-08}
https://{site_url}/wp-json/my-api/v1/events/create/
https://{site_url}/wp-json/my-api/v1/events/update/{event_id}

== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `Event Booking` and click 'Install Now' or click on the `upload` link to upload `event-booking.zip`
4. Click on `Activate plugin`
